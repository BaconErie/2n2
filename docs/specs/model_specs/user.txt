USER MODEL

This model represents a user.

Attributes:
- user_id, string
- username, string
- password, string: Hashed with salt
- salt, string
- created_study_guides, list of StudyGuides: A list of StudyGuide objects that the user has created
- recently_studied, list of StudyGuides: A list of StudyGuide objecs that the user has recently studied, max length 6
- date_created, int: UTC Unix timestamp of when the user was created
