USER MODEL

This model represents a user.

Attributes:
- id, int
- username, string
- password, string: Hashed with salt
- salt, string
- study_guide_set, list of StudyGuides: A list of StudyGuide objects that the user has created. Note that Django will create this for us.
- recently_studied, list of int: A list of integers which represent StudyGuide ids. 
- date_created, int: UTC Unix timestamp of when the user was created
